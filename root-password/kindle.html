Amazon Kindle root password tool

This page will tell you your Kindle 4's default root password.

Enter the settings page and look in the bottom right corner to determine your firmware version. You'll also see the serial number in the settings.
Enter your kindle serial number below and it will compute your password.
It's all done locally in your browser using javascript, no information is transferred away from your computer.

There is another login on most Kindles: User framework with password mario.

I have received a report of a Kindle with 4.0 firmware that uses "fionaXXX" as password and a 4.0.1 firmware device with "mario" as password, so if one doesn't work, try the other.

This script was converted from a little python script to javascript. It uses a BSD-licensed MD5-Javascript library by Paul Johnston.
